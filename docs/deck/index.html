SLIDE_05 // CLAUDE — THREE VOICES

Same data. Three audiences. One Claude pass per voice.

The evidence block is identical in every prompt — only the system message changes. Sonnet 4.6, with strict factual constraints (no hallucinated buyers, no rounding into different orders of magnitude).

// EXECUTIVE_BRIEF

For a Microsoft sustainability lead.

"Risk Rating: MODERATE. Project demonstrates strong additionality vs controls (+25 pp), but a 2016 single-year loss spike of 1,233.5 ha equals 28% of all post-start loss and warrants verification before further credit retirements."
~550 words · risk pill · timeline · recommended action
// INVESTIGATIVE_ARTICLE

For a Guardian environment desk.

"Amazon Carbon Credit Project Shows Striking Forest Protection — But a Volatile Middle Decade Raises Questions. The most striking number isn't how much forest was saved; it's what happened in 2016."
~640 words · narrative · per-year timeline · attribution
// REGULATOR_BRIEF

For a Verra compliance auditor.

"VCS Standard 3.4 (Additionality) — SATISFIED on aggregate (project rate 2.12% vs control mean 27.1%). VCS Standard 3.14 (Monitoring) — flagged: 2016 in-boundary spike requires explanatory documentation against monitoring plan."
~1,000 words · cited sections · table · action plan
